Global Variable: isDarkMode: A boolean to keep track of whether the dark mode is active.

  1. Function change2():
    • Purpose: Randomly changes the background color of the body.
    • It defines an array of color codes.
    • It selects a random color from the array using Math.random() and sets the body's backgroundColor property to that color.
  2. Function changeMode():
    • Purpose: Toggles between light and dark modes.
    • It toggles the isDarkMode variable to switch states.
    • Based on the value of isDarkMode, it sets the body's background and text color:
      • Dark mode: dark background with light text.
      • Light mode: light background with dark text.
